# lenis45.github.io — Personal Portfolio

Personal portfolio of **Denis Kolesnikov** (AI Engineer), served via GitHub Pages.

**Live:** https://lenis45.github.io

## Stack
Static site — plain HTML, CSS and JavaScript, no build step.

- `index.html` — entry page
- `css/` — styles
- `js/` — scripts
- `assets/` — images and media

## Run locally
```bash
python3 -m http.server 8000   # open http://localhost:8000
```
Deployment is automatic from the `main` branch via GitHub Pages.
